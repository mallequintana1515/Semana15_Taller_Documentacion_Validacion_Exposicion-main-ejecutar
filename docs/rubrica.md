# Rubrica de evaluacion

| Criterio | Superior | Alto | Basico | Bajo |
|---|---|---|---|---|
| Proyecto ejecutable | Instala y ejecuta correctamente; pagina y JSON Server funcionan | Ejecuta con ajustes menores | Ejecuta parcialmente | No ejecuta |
| Documentacion tecnica | Explica claramente objetivo, tecnologias, instalacion, ejecucion, estructura y pruebas | Explica la mayoria de secciones | Tiene informacion incompleta | No documenta adecuadamente |
| Bitacora de validacion | Registra minimo 5 pruebas con resultados, errores, correcciones y evidencias | Registra pruebas claras pero con algunos vacios | Registra pocas pruebas o muy generales | No evidencia proceso de validacion |
| Correcciones aplicadas | Corrige errores y justifica cambios en archivos concretos | Aplica correcciones funcionales | Aplica cambios limitados | No corrige o no justifica |
| Evidencias | Presenta capturas organizadas y relacionadas con las pruebas | Presenta evidencias suficientes | Presenta pocas evidencias | No presenta evidencias claras |
| Guion de exposicion | Organiza una exposicion tecnica clara, demostrable y reflexiva | Presenta un guion comprensible | Guion incompleto | No prepara exposicion |
| Organizacion | Carpetas y archivos ordenados | Estructura mayormente ordenada | Estructura confusa | Entrega desordenada |
