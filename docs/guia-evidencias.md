# Guia para organizar evidencias

Guarda tus capturas dentro de la carpeta `evidencias/`. Usa nombres claros.

## Carpetas sugeridas

| Carpeta | Que guardar alli |
|---|---|
| evidencias/consola | Capturas de consola sin errores criticos |
| evidencias/pruebas | Capturas de formulario, tabla, grafico y datos guardados |
| evidencias/responsive | Capturas de la vista movil |
| evidencias/antes-despues | Capturas que comparen un problema y su correccion |
| evidencias/exposicion | Capturas o material usado para la exposicion |

## Recomendacion de nombres

Usa nombres como:

```text
01-proyecto-abierto.png
02-formulario-vacio-error.png
03-registro-guardado-json-server.png
04-consola-sin-errores.png
05-vista-responsive.png
06-antes-validacion-telefono.png
07-despues-validacion-telefono.png
```

Una evidencia debe permitir entender que probaste y cual fue el resultado.
