# Glosario rapido

| Termino | Significado |
|---|---|
| Documentacion tecnica | Explicacion organizada de como funciona, se instala y se ejecuta un proyecto |
| README | Archivo principal de informacion de un proyecto |
| Bitacora | Registro ordenado de pruebas, errores, correcciones y evidencias |
| Validacion | Proceso de comprobar que los datos o acciones cumplen reglas definidas |
| Correccion | Cambio aplicado para solucionar un error encontrado |
| Evidencia | Captura o prueba que demuestra un resultado |
| Consola | Herramienta del navegador donde se ven errores y mensajes del codigo |
| JSON Server | Herramienta que simula una base de datos local usando un archivo JSON |
| API | Medio por el cual una aplicacion consulta o guarda datos |
| Responsive | Capacidad de una pagina para adaptarse a diferentes tamanos de pantalla |
| Exposicion tecnica | Presentacion donde se explica el proyecto, su funcionamiento, pruebas y decisiones |
